# NVD

[취약점 찾기 셀]
OSV (x), trivy를 통해 다양한 소스의 실제 취약점 데이터를 기반으로 정확한 CVSS 점수를 매김.

[동적 검사 데이터 셀]
동적 특징 4가지를 추출, AI가 공정하게 비교하도록하는 AI 모델
standardscaler로 스케일링 작업을 한다.
pid를 가지고 실행 중인지, 정상 경로인지, 자원을 얼마나 쓰는지 검사한다.

--> 수정 사항1: 데이터 유출 방지를 위해 순서 변경

--> 수정 사항2: 올바른 스케일링을 위해 연속형 특징에만 스케일링.

[오토인코더 모델 셀]
정상과 비정상 구분해 제로 트러스트를 검증하는 과정.

--> 수정 사항1: 데이터 유출 문제를 위해 위의 순서대로 검증 절차를 거치도록 수정

--> 수정 사항2: 최종 출력 sigmoid은 무조건 0과 1사이 값만 만듦. 그러나, 모델에 입력한 데이터는 standardscaler를 거쳐 평균이 0, 표준편차가 1인 값이다. 그러므로, 입력 데이터와 출력 데이터 범위가 안 맞기 때문에, 최종 출력을 linear로 변경.

--> 수정 사항3: 임의 기준 말고, 비지도 방식에 맞는 데이터로 가정하여 수정

--> 수정 사항4: 결과 예측 수행을 위해, 스케일링된 데이터가 필요.
